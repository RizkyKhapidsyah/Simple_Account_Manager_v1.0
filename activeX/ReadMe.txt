Source code program menggunakan activex control eksternal (ocx), sebelum menjalankan source code, terlebih dahulu dilakukan registrasi activex control (user control) / ocx ke dalam sistem operasi agar library user control dapat digunakan dan berjalan dengan baik. Untuk registrasi secara otomatis dilakukan melalui file 'install.bat' pada folder ini menggunakan melalui command prompt Windows dengan mode Administrator. 

Jika registrasi gagal, lakukan manual melalui baris perintah di command prompt dan arahkan ke direktory folder ini, setelah itu jalankan file 'install.bat'
Registrasi file user control dilakukan agar source code berjalan dengan baik dan tidak ada error. Namun jika registrasi masih gagal, silahkan registrasi activex control (ocx) secara manual melalui regsvr32 milik sistem operasi Windows.
